Over the past week we have been learning how to use the jinja2 templating library instead of python3's string.format methods for handling templating in our static site generators. I am very impressed with how straight forward jinja2 is when it comes to templating our code. Even setting up inheritance between two different tempaltes was relativity straightforward once determining how to load correct template files. All in all, I would recommend learning this library if doing any sort of advanced templating.
